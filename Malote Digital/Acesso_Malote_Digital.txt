ID: 
GW-003 

Sistema: Malote Digital
Categoria: Malote Digitl
Procedimento: Criação de usuário no Malote Digital

Objetivo:
Criar um usuário dentro do Malote digital

Quem pode solicitar:
- Chefia da unidade/Chefe do servidor ou estagiário

Quem executa:
- Equipe 1N Administração de Usuário

Pré-requisitos:
- Usuário deve possuir E-mail institucional ativo. O Malote não permite o cadastro de E-mail pessoal.

Informações necessárias:
- Nome completo
- CPF

Prazo:
24h

Fluxo:
1. Chefia solicita.
2. Equipe 1N Admin verifica que usuário possui e-mail institucional
3. Equipe 1N Admin cria o usuário.
4. O sistema envia automaticamente a senha para o e-mail do usuário.

Observações:
O login deve ser padronizado para ser sempre o login de rede com letras minúsculas.

Palavras-chave:
- Criar acesso Malote
- Malote Digital
